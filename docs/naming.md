# Naming Conventions

## CSS

- `.component__container` - for a wrapper
- `.component` - for the main class
- `.component__block`, `.component__block__element` - for sub-classes
- `.is_active` - for modifiers

Naming is inspired by:

- http://getbem.com/naming/
- https://thesassway.com/modular-css-naming-conventions/
